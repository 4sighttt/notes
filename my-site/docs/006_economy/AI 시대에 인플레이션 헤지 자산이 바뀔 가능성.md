**핵심 요약**

1. AI 시대에는 인플레이션 구조 자체가 바뀔 가능성이 있다.
    
2. 과거 인플레이션: **에너지·원자재 부족 → 물가 상승**.
    
3. AI 시대 인플레이션: **전력·컴퓨팅·데이터 인프라 부족** 가능성.
    
4. 따라서 새로운 헤지 자산으로 **전력, 데이터센터, 반도체 인프라**가 주목된다.
    
5. 즉 “원자재 중심 인플레 헤지 → 인프라 중심 헤지”로 이동 가능성이 있다.

# 정리

AI 시대 인플레이션 구조

```
에너지 인플레
+ 컴퓨팅 인플레
+ 데이터 인프라 인플레
```

따라서 인플레이션 헤지 자산도

```
금/원자재 중심
→ 전력/데이터센터/반도체 중심
```

으로 일부 이동할 가능성이 있다.

---

원하면 다음도 설명할 수 있다.

1️⃣ **AI 때문에 실제로 전력 부족이 발생하는 이유**  
2️⃣ **AI 데이터센터가 전 세계 전력 소비를 얼마나 늘릴지**  
3️⃣ **AI 시대에서 가장 중요한 자산 5가지**.


**핵심 요약**

1. AI 경제에서 가장 중요한 자산은 **지능 생산에 필요한 인프라**다.
    
2. 핵심 5가지: **전력, 반도체, 컴퓨팅 인프라, 데이터, 인재**.
    
3. AI는 단순 소프트웨어가 아니라 **거대한 산업 시스템**이다.
    
4. 따라서 “AI 기업”보다 **AI 인프라 기업**이 더 중요할 수 있다.
    
5. 실제 투자 흐름도 **전력·데이터센터·반도체 장비**로 이동 중이다.

# AI 시대 핵심 자산 5가지

## 1️⃣ 전력 (Electricity)

AI 데이터센터는 엄청난 전력을 소비한다.

예

- GPT급 모델 학습
    
- 대형 AI 클러스터
    

구조

```
AI
↓
GPU
↓
전력
```

즉

```
AI = 전력 산업
```

전력 기업 예

- NextEra Energy
    
- Duke Energy
    

---

## 2️⃣ 반도체 (Semiconductors)

AI 연산 핵심

```
GPU
AI 가속기
HBM 메모리
```

대표 기업

- NVIDIA
    
- TSMC
    
- ASML
    

AI 붐의 핵심 병목은

```
GPU 공급
```

이다.

---

## 3️⃣ 컴퓨팅 인프라 (Data Centers)

AI는 서버 산업이다.

구조

```
데이터센터
GPU
네트워크
냉각
```

대표 기업

- Equinix
    
- Digital Realty
    

AI 확산 → 데이터센터 수요 폭발.

---

## 4️⃣ 데이터 (Data)

AI 학습 구조

```
데이터
→ 모델 학습
→ 지능
```

데이터가 많을수록

```
모델 성능 ↑
```

그래서 중요한 기업

- Alphabet
    
- Meta Platforms
    

이들은 **세계 최대 데이터 보유 기업**이다.

---

## 5️⃣ 인재 (Talent)

AI 연구는 아직 **인간 연구자 의존도**가 높다.

핵심 인재

- 모델 연구자
    
- 시스템 엔지니어
    
- 반도체 설계자
    

대표 연구 조직

- OpenAI
    
- DeepMind
