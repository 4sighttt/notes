(a+b)(b+c)(c+a)+abc를 인수분해 하면?


[  
(a+b)(b+c)(c+a)+abc  
]

여기서

[  
x=a+b+c  
]

로 두자.

그러면


$a+b=x-c,\quad b+c=x-a,\quad c+a=x-b$

이므로 원식은

[  
(x-c)(x-a)(x-b)+abc  
]

가 된다.

이제 ((x-a)(x-b))부터 계산하면

[  
$(x-a)(x-b)=x^2-x(a+b)+ab$
]

그런데 (a+b=x-c) 이므로

[  
$=x^2-x(x-c)+ab=xc+ab$  
]

따라서 원식은

[  
(x-c)(xc+ab)+abc  
]

가 된다.

전개하면

[  
=x(xc+ab)-c(xc+ab)+abc  
]

[  
$=x^2c+xab-xc^2-abc+abc$  
]

[  
$=x^2c+xab-xc^2$  
]

여기서 (x)를 묶으면

[  
$=x(xc+ab-c^2)$
]

이제 괄호 안을 정리하자.

[  
$xc+ab-c^2=cx-c^2+ab=c(x-c)+ab$  
]

그런데

[  
x-c=a+b  
]

이므로

[  
=c(a+b)+ab=ac+bc+ab  
]

결국

[  
(a+b)(b+c)(c+a)+abc=x(ab+bc+ca)  
]

마지막으로 (x=a+b+c)를 다시 대입하면

[  
${(a+b+c)(ab+bc+ca)}$
]

이다.
