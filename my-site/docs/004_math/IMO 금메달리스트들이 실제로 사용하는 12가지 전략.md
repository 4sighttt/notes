**핵심 요약**

1. 국제수학올림피아드(IMO) 금메달리스트들은 문제를 바로 풀기보다 **구조 탐색 전략**을 사용한다.
    
2. 가장 중요한 전략은 **문제 단순화, 패턴 탐색, 불변량 발견, 표현 변경**이다.
    
3. 많은 전략은 **George Pólya**의 문제 해결 방법과 올림피아드 훈련 자료에서 공통적으로 등장한다.
    
4. 핵심은 계산 능력보다 **문제 구조를 발견하는 능력**이다.
    
5. 아래 전략들은 IMO 훈련 자료와 수학 교육 연구에서 반복적으로 언급되는 방법들이다.

# IMO 금메달리스트들이 자주 사용하는 12가지 전략

## 1. 작은 사례 실험

가장 기본적인 전략.

예

```
n = 1
n = 2
n = 3
n = 4
```

작은 사례에서

- 패턴
    
- 규칙
    
- 예외
    

를 찾는다.

---

## 2. 문제 단순화

문제를 더 쉬운 버전으로 만든다.

예

```
일반 문제 → 특수한 경우
```

예시

- 변수 개수 줄이기
    
- 차원 축소
    
- 대칭 상황
    

---

## 3. 그림 또는 시각화

특히 다음 분야에서 중요하다.

- 기하
    
- 조합론
    
- 그래프 이론
    

그림을 그리면 **숨은 관계**가 드러난다.

---

## 4. 불변량 찾기 (Invariant)

어떤 과정에서도 **변하지 않는 값**을 찾는다.

예

- 짝수 / 홀수
    
- 합
    
- 대칭
    

많은 올림피아드 문제의 핵심 아이디어다.

---

## 5. 단조성 찾기 (Monovariant)

반복 과정에서 어떤 값이

```
계속 증가하거나 감소
```

하는 경우를 찾는다.

이를 통해

- 과정 종료
    
- 불가능성
    

을 증명할 수 있다.

---

## 6. 극단적 경우 분석

문제의 경계 상황을 본다.

예

- 최소값
    
- 최대값
    
- 특정 값
    

이 방법은 **문제 구조를 드러낸다.**

---

## 7. 역으로 생각 (Backward reasoning)

목표에서 출발한다.

```
목표 → 필요한 조건 → 이전 단계
```

특히 증명 문제에서 매우 강력하다.

---

## 8. 문제 변형

원 문제를 살짝 바꿔본다.

예

- 조건 제거
    
- 조건 추가
    
- 더 일반적인 문제
    

이 과정에서 **핵심 아이디어**가 보인다.

---

## 9. 대칭성 찾기

많은 올림피아드 문제는 **대칭 구조**를 가진다.

예

- 변수 교환
    
- 기하 대칭
    
- 순환 구조
    

대칭을 이용하면 계산이 크게 줄어든다.

---

## 10. 귀납적 사고

패턴을 발견한 뒤

```
작은 사례 → 일반 증명
```

으로 확장한다.

---

## 11. 보조 구조 만들기

새로운 요소를 추가한다.

예

- 보조선
    
- 보조점
    
- 새로운 변수
    

특히 기하 문제에서 매우 중요하다.

---

## 12. 여러 표현 사용

같은 문제를 다른 언어로 본다.

예

```
대수 ↔ 기하
조합 ↔ 그래프
수열 ↔ 함수
```

표현이 바뀌면 **새로운 구조**가 보인다.

# 가장 자주 등장하는 5가지 올림피아드 아이디어

## 1. 불변량 (Invariant)

어떤 과정이 반복되어도 **변하지 않는 값**을 찾는 방법.

예

- 짝수 / 홀수
    
- 합
    
- 색깔 패턴
    
- parity
    

구조

```
상태 변화 → 어떤 값은 항상 동일
```

이 아이디어는 특히

- 게임 문제
    
- 조합 문제
    

에서 매우 자주 등장한다.

---

## 2. 단조량 (Monovariant)

어떤 값이 과정에서 **계속 증가하거나 감소**하는 경우.

```
상태 → 값이 항상 증가
또는
상태 → 값이 항상 감소
```

이것을 이용하면

- 과정이 반드시 끝남
    
- 특정 상태가 불가능함
    

을 증명할 수 있다.

---

## 3. 대칭 (Symmetry)

문제 속에 숨어 있는 **대칭 구조**를 활용하는 방법.

예

- 변수 교환
    
- 순환 구조
    
- 기하 대칭
    

대칭을 발견하면

```
여러 경우 → 하나의 경우
```

로 줄일 수 있다.

---

## 4. 극단 원리 (Extremal Principle)

어떤 집합에서 **가장 크거나 가장 작은 요소**를 선택해 분석한다.

예

- 가장 긴 선분
    
- 가장 큰 수
    
- 최소 값
    

이 극단적인 대상은 종종 **특별한 성질**을 가진다.

---

## 5. 구성 (Construction / Auxiliary element)

문제 해결을 위해 **새로운 요소를 만들어 추가**하는 방법.

예

- 기하에서 보조선
    
- 새로운 변수
    
- 새로운 집합
    

구조

```
원래 문제
↓
새 구조 추가
↓
숨은 관계 드러남
```

특히 기하 문제에서 매우 자주 등장한다.

---

# 올림피아드 아이디어의 핵심 특징

이 다섯 가지 전략의 공통점은 다음이다.

```
문제 계산 → 구조 발견
```

즉 올림피아드 수학은

- 계산 능력
    
- 공식 암기
    

보다

**문제의 구조를 보는 능력**을 요구한다.
