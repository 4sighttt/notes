[[우리는 현실 세계에 살고 있는 것이 아니다]]

## 핵심 요약

- **벨의 부등식**은 “입자의 성질이 측정 전부터 이미 정해져 있고(실재성), 먼 곳은 즉시 영향을 주지 않는다(국소성)”는 가정이 맞다면 반드시 만족해야 하는 통계적 제약이다.
    
- 실제 **얽힘 실험**에서는 이 제약이 반복적으로 **위반**된다.
    
- 따라서 최소한 **두 가정 중 하나 이상은 틀려야 한다.**
    
- 그래서 물리학자들은 보통 다음 **세 가지 해석 방향** 중 하나를 택한다.
    
- (1) 비국소성 인정
    
- (2) 측정 전 성질이 없다는 해석(실재성 수정)
    
- (3) 초결정론(측정 선택도 미리 결정됨)
    
- 즉 “현실이 없다”기보다 **고전적 현실 개념이 그대로 성립하지 않는다**는 결론이다.
    

---

# 1. 벨의 부등식 직관 설명 (수식 없이)

### 기본 상황

서로 **얽힌 두 입자**를 만든 뒤  
멀리 떨어진 두 실험실에서 각각 측정한다.

예

- Alice: 왼쪽 실험실
    
- Bob: 오른쪽 실험실
    

둘은 **측정 방향을 임의로 선택**할 수 있다.

---

### 아인슈타인의 가정

아인슈타인은 두 가지를 믿었다.

1️⃣ **실재성 (realism)**  
입자의 성질은 측정 전부터 이미 존재한다.

2️⃣ **국소성 (locality)**  
멀리 떨어진 곳은 즉시 영향을 줄 수 없다.

즉

```
Alice의 측정 결과
= 입자가 원래 가지고 있던 값

Bob의 측정 결과
= 그 입자가 원래 가지고 있던 값
```

서로 **빛보다 빠르게 영향 줄 수 없음**

---

### 벨의 아이디어

존 벨은 질문을 바꿨다.

> 만약 아인슈타인이 맞다면  
> 결과들의 **상관관계 패턴**에 제한이 생겨야 한다.

즉

```
가능한 통계 패턴의 범위
```

가 **수학적으로 제한**된다.

이 제한이 바로

**벨의 부등식**

---

### 실험 결과

얽힌 입자를 실험해 보면

```
실험 결과 상관관계
> 벨의 부등식이 허용하는 범위
```

즉

**고전적 설명으로는 불가능한 상관관계**

가 나타난다.

이 실험을 대표적으로 수행한 사람이

- 알랭 아스페
    
- 존 클라우저
    
- 안톤 차일링거
    

→ **2022 노벨물리학상**

---

# 2. 그래서 남는 선택지

벨 실험이 말하는 것

```
국소성 + 실재성
둘 다 동시에 유지 불가능
```

그래서 물리학에는 크게 **3가지 선택지**가 있다.

|선택지|의미|대표 해석|
|---|---|---|
|비국소성|멀리 떨어진 입자가 즉시 연결|보흠 해석|
|실재성 포기|측정 전에는 성질이 없음|코펜하겐|
|초결정론|측정 선택도 미리 결정|초결정론|

---

## ① 비국소성

얽힌 입자는 사실상 **하나의 시스템**

```
Alice 측정
→ Bob 결과와 즉시 상관
```

하지만

```
정보 전달은 불가능
```

그래서 상대성이론과 직접 충돌하지는 않는다.

대표 해석

**보흠 해석 (Bohmian mechanics)**

---

## ② 실재성 수정 (코펜하겐)

입자의 성질은

```
측정 전
= 확률 상태
```

측정 순간

```
파동함수 붕괴
```

→ 결과가 결정

즉

```
측정이 현실을 만든다
```

라는 해석.

---

## ③ 초결정론

가장 급진적 해석.

```
실험자가 선택한 측정 방향조차
우주 초기부터 이미 결정됨
```

즉

```
측정 선택
입자 상태
```

둘 다 **공통 원인**이 있다.

그래서 벨 부등식 위반도 설명 가능.

하지만

```
자유의지 문제
```

때문에 많은 물리학자가 꺼린다.

---

# 3. 중요한 오해 하나

벨의 부등식이 말하는 것은

❌ 현실이 없다  
❌ 모든 것이 환상이다

가 아니다.

정확한 의미는

```
현실은 존재하지만
고전 물리학이 가정했던 방식으로
존재하지 않는다
```

이다.

---

원하면 다음도 설명해 줄 수 있다.

1️⃣ **왜 얽힘 상관관계는 빛보다 빠른 통신이 불가능한지**  
2️⃣ **벨의 부등식을 실제 예시(양말 상자 모델)로 직관적으로 설명**  
3️⃣ **왜 아인슈타인은 끝까지 양자역학을 받아들이지 않았는지**

이 세 가지는 양자역학 이해에서 **가장 중요한 부분**이다.
